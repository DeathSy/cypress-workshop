## Initial setup

```sh
 $ git clone https://github.com/DeathSy/cypress-workshop.git
 $ cd cypress-workshop && yarn install
```

## Environment setup (Mac)

✅ git
✅ nodeJS v10.16 or above
✅ yarn v1.17 or above
✅ VSCODE (recommend)

```sh
$ git --version
$ node --version
$ yarn --version
```
